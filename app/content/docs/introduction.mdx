## Architecture

Go2 is built entirely on Cloudflare's developer platform:

- **Cloudflare Workers** - Serverless functions running at the edge
- **Cloudflare KV** - Low-latency key-value storage for fast lookups
- **Cloudflare D1** - SQLite database for link and user management
- **Analytics Engine** - Real-time click analytics at scale

This architecture means zero cold starts, automatic scaling, and global performance.
